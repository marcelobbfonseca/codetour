Proposta

A proposta do curso é criar independencia

Aprendendo a aprender100%
Lógica de programação100%
Construindo o seu projeto100%

Esse curso é para você que quer aprender a programar ou desenvolver suas skills mas voltado para algo de seu interesse.

Nosso objetivo não é te jogar conteúdo. Queremos ensinar o caminho das pedras para as pessoas a aprenderem a poder se virar. Em programação você nunca para de aprender. Os desenvolvedores de software trabalham aprendendo e em seguida codificando.

Não se aprende linguagens de programação, e sim aprendemos a programar. Com esse princípio, desenvolvemos a lógica de programação. Com esta habilidade na ponta dos dedos, podemos programar em qualquer linguagem.
